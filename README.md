# ChatGPT For SlackBot

ChatGPT 機能の SLACK BOT
[参考](https://manchann.tistory.com/43)

### 1. 必須 library

```
pip install -r requirements.txt
```

### 2. OpenAI API Key 記入

chatgpt.py

```
YOUR_API_KEY = '<your-openai-api-key>'
```

### 3. Slack Bot user token 記入

main.py

```
bot_token = "<your-slack-bot-token>"
```

### 4. code アクセス

```
python main.py
```
